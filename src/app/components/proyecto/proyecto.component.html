Portfolio web personalizable

Desarrollé un portfolio web que permite a los usuarios personalizar su información de nombre, estudio y experiencia laboral de manera intuitiva y fácil. Además, implementé funcionalidades de autenticación y autorización para asegurar la privacidad y seguridad de los usuarios. Para la implementación de la aplicación utilicé tecnologías de última generación como Spring Boot, Java y Angular, lo que permitió una integración fluida entre el back-end y el front-end. También implementé una arquitectura de base de datos escalable para gestionar la información de los usuarios de manera eficiente.



Sistema de ventas y reportes de empleados en C++

Desarrollo de un sistema para la gestión de ventas de una empresa que incluye la generación de reportes de empleados y productos vendidos. El sistema está desarrollado en C++ y utiliza un archivo de empleados y otro de ventas para almacenar la información. Se utiliza un vector estático para guardar los empleados y pilas dinámicas para cada empleado que contienen los productos vendidos por cada uno de ellos. El sistema genera reportes ordenados por recaudación de cada empleado y los productos vendidos por fecha de venta descendente.
